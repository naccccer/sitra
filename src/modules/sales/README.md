# Sales Frontend Module

Owns:
- Order creation/edit flows
- Orders list and workflow UI
- Sales-specific page entry points

Public entry points:
- `pages/OrderCreatePage.jsx`
- `pages/OrderDetailPage.jsx`
- `pages/OrdersPage.jsx`
- `components/customer/OrderForm.jsx`
- `components/admin/AdminOrdersView.jsx`

Compatibility:
- Legacy paths in `src/pages/*` and `src/components/*` re-export from this module.
